zurb-foundation-wysihtml5

Simple, beautiful wysiwyg editors

About

zurb-foundation-wysihtml5 is a straightforward port based on jhollingworth's brilliant bootstrap-wysihtml5. this is a javascript plugin that makes it easy to create simple, beautiful wysiwyg editors with the help of wysihtml5 and Zurb Foundation

View project on Github

Usage

$('.textarea').wysihtml5();

Dependencies

  1. wysihtml5
  2. Zurb Foundation
  3. foundation glyphicons
